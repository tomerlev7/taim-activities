var HEADERS=['תא','פעילות','תיאור','יום','תאריך','שעה','מיקום','צרכים ומשאבים','תאריך דיווח']; function getWeekSheetName(dateStr){ var d=new Date(dateStr); var day=d.getDay(); var diff=day===0?0:-day; var sun=new Date(d);sun.setDate(d.getDate()+diff); var dd=String(sun.getDate()).padStart(2,'0'); var mm=String(sun.getMonth()+1).padStart(2,'0'); return 'שבוע '+dd+'.'+mm+'.'+sun.getFullYear(); } function getOrCreateSheet(name){ var ss=SpreadsheetApp.getActiveSpreadsheet(); var sheet=ss.getSheetByName(name); if(!sheet){ sheet=ss.insertSheet(name); sheet.appendRow(HEADERS); sheet.setRightToLeft(true); sheet.getRange(1,1,1,HEADERS.length) .setFontWeight('bold').setBackground('#E1F5EE'); sheet.setFrozenRows(1); } return sheet; } function doPost(e){ var data=JSON.parse(e.postData.contents); data.forEach(function(r){ var sheetName=r.date?getWeekSheetName(r.date):'שבוע כללי'; var sheet=getOrCreateSheet(sheetName); sheet.appendRow([r.cell,r.title,r.desc,r.day, r.dateFormatted||r.date,r.time,r.location,r.needs, r.reported||new Date().toLocaleDateString('he-IL')]); }); return ContentService.createTextOutput( JSON.stringify({status:'ok'})) .setMimeType(ContentService.MimeType.JSON); } function doGet(e){ var ss=SpreadsheetApp.getActiveSpreadsheet(); var props=PropertiesService.getScriptProperties(); if(e&&e.parameter&&e.parameter.setLock){ props.setProperty('form_locked',e.parameter.setLock); return ContentService.createTextOutput(JSON.stringify({ok:true})) .setMimeType(ContentService.MimeType.JSON); } if(e&&e.parameter&&e.parameter.getLock){ var locked=props.getProperty('form_locked')==='true'; return ContentService.createTextOutput(JSON.stringify({locked:locked})) .setMimeType(ContentService.MimeType.JSON); } var sheets=ss.getSheets(); var week=e&&e.parameter&&e.parameter.week; var result=[]; sheets.forEach(function(sheet){ if(sheet.getName().indexOf('שבוע')!==0)return; if(week&&sheet.getName()!==week)return; var data=sheet.getDataRange().getValues().slice(1); data.forEach(function(r){ if(r[0])result.push({cell:r[0],title:r[1],desc:r[2], day:r[3],date:r[4],time:r[5],location:r[6], needs:r[7],reported:r[8],sheet:sheet.getName()}); }); }); return ContentService.createTextOutput(JSON.stringify(result)) .setMimeType(ContentService.MimeType.JSON); }
